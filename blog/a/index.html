First Blog

This is my first blog on the site

I don’t know exactly if I will be posting regularly on the blogs. I am sure not many people are eagerly waiting either. However since I used Hugo to create this site I though why not make a tutorial for new users who might want their own website.

Installation

I installed hugo on Artix Linux so it was as easy as

> pacman -S hugo

You can find hugo on most of the distros and their respective repos.

After that follow hugo’s quick start guide they explain it better than I ever will but come back to this page after you are done there are some small things that need to be done to have a fully functional site.

Setup

Your theme might provide you with enough documentation to completely setup your website but since you returned I would assume it didn’t Hugo’s quick start tutorial ends after creating the first blog. Now what if you want to customize the homepage. What you need to understand are the basics of hugo. Hugo works on templates that automatically get filled with the content from the markdown files your created.

Open the theme folder and look at the structure you will find that it similar to the parent hugo directory.

Go to hugo folder then themes > “Your theme name” > layouts.

You will see two folders “_default” and “partials”

  • partials contains snippets of code that are being called by files in “_default”.

Go to _default and you should see three files

baseof.html  list.html  single.html

If you don’t see baseof file don’t panic its fine. Hugo assume that there are two kind of pages one’s a list and the other a single page. List is where you see your blog titles in a list and links to various other pages and single is when you open the page of a particular post. For example currently you are on a single page but when you click on blog you will land to a list page.

If you still don’t get it watch this, a playlist on hugo by Mike Dane. You can skip first three episode start from 4th go till 16th.

Two things you might want to do is create an about page and a special homepage

  • copy single.html from the _default folder to hugo’s parent directory layouts folder.
  • Name it index.html and you have your homepage now to edit it you need to know html.

Secondly

  • Create a separate folder in content folder with any name and a file.
hugo new name/name.md

This will create folder name with file name.md

  • now create folder with same name in layouts directory

This you can edit directly by editing the markdown file created although you can always change the HTML Layout.

Remember we are making these folders in hugo’s parent folder not the theme folder.

Configuration

You will find config.toml at the base of hugo directory. Open it and fill baseURL = “your website url”. You don’t need this for local viewing but if you plan on hosting the website it is required. Change the title to whatever you want. To add items in the navigation pane of your theme

[[menu.main]]
name = "Name for the option"
url = "URL/the respective directories"
weight = n, where n is a number

Weight is used to decide the sequence of the options. Your url field would contain the link to your website and then the folder if its a list page or folder and then the file if its a single page For eg.

[[menu.main]]
name = "Blog"
url = "https://mywebsite.whatever/blog/
weight = 1

This will give me a option to go the the blog page

[[menu.main]]
name = "Blog"
url = "https://mywebsite.whatever/name/name
weight = 1

This will open the name markdown you created Remember don’t end the url with .md just use the file name

Finally if you want to add some pictures in any page store the pictures in static/somefolder and refer to them with path /somefolder/img.jpg. They won’t display on you local server but they will online on your website.
